# Changelog

All notable changes to this project will be documented in this file.

## v1.0.0

### Added
- Initial open source release of GC-Writer Assistant
- Chapter import and chapter list display
- Chapter content display and linked reading area
- Chapter outline extraction workflow
- Writing workspace with chapter management
- Local folder selection and chapter file saving
- AI polish feature
- AI continuation feature
- Storyline summary feature
- Theme configuration with light, dark, and custom theme support
- Windows startup script for beginner-friendly launch

### Improved
- Responsive layout optimization for major panels
- Right-side multi-panel workflow and interaction refinement
- Writing workspace persistence and local save flow
- README instructions for beginner installation and usage
