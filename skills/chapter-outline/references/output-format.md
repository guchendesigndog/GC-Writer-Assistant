# Output Format

Each selected chapter should render like this:

```text
第1章
1. 第一条主线要点
2. 第二条主线要点
3. 第三条主线要点
```

Rules:

- Keep the three points centered on the chapter's important main line
- Total length should be around 200 Chinese characters
- Prefer concrete plot movement over abstract evaluation
- Keep numbering stable as `1. 2. 3.`
