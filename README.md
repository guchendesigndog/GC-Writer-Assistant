# GC-Writer Assistant

GC-Writer Assistant 是一个面向中文网文创作场景的本地化作家助手写作工具，支持章节导入、章节展示、章节大纲提取、码字、AI 润色、AI 续写、主线框架整理、本地文件夹保存以及主题配置等功能。

当前版本适用于 **Windows**。

---

## 功能简介

- 导入常见小说文本文件
- 自动识别章节并生成章节列表
- 章节展示区与码字区联动参考
- 提取章节大纲
- AI 润色
- AI 续写
- 当前主线框架整理
- 亮色 / 暗色 / 自定义主题配置
- 本地文件夹保存章节内容
- 自动记忆当前写作状态

## 安装前准备

如果你的电脑是新电脑，还没有开发环境，请先完成下面两步。

### 1. 安装 Node.js

请先安装 Node.js 20 版本。

下载地址：

[https://nodejs.org/](https://nodejs.org/)

安装完成后，打开 `PowerShell` 或 `cmd`，输入：

```bash
node -v
npm -v
```

如果能看到版本号，说明安装成功。

### 2. 安装浏览器

请安装以下任意一种最新版浏览器：

- Chrome
- Edge

建议优先使用这两种浏览器，因为本项目需要用到本地文件夹访问能力。

## 下载项目

你可以任选一种方式获取项目。

### 方式一：Git 克隆

```bash
git clone <你的仓库地址>
cd "GC-Writer Assistant"
```

### 方式二：下载 ZIP

1. 在 GitHub 页面点击 `Code`
2. 点击 `Download ZIP`
3. 解压到本地目录
4. 打开终端进入项目目录

例如：

```bash
cd "D:\GC-Writer Assistant"
```

---

## 启动项目

在项目根目录，你可以使用下面两种方式启动项目。

### 方式一：双击一键安装启动（推荐新手）

项目根目录内提供了启动脚本：

```text
一键安装启动.bat
```

双击后会自动执行以下操作：

- 检查电脑是否已安装 Node.js
- 检查 npm 是否可用
- 如果当前项目还没有安装依赖，会自动执行 `npm install`
- 自动启动项目

启动成功后，在浏览器打开：

```text
http://localhost:3000
```

### 方式二：双击快速启动

如果你已经完成过依赖安装，也可以直接双击：

```text
start.bat
```

这个脚本会直接启动项目，不负责自动安装依赖。

### 方式三：手动命令启动

如果你熟悉命令行，也可以在项目根目录执行：

```bash
npm install
npm run dev
```

启动成功后，在浏览器打开：

```text
http://localhost:3000
```

## 首次使用

### 1. 配置 AI 接口

打开项目后，进入右下角 **设置** 页面，填写：

- 提供商
- 端点地址
- 模型名称
- API Key

未配置接口时，AI 润色、AI 续写、主线框架等功能无法使用。

### 2. 选择本地保存文件夹

进入 **码字** 面板后，点击：

```text
选择本地文件夹
```

选择你自己的小说保存目录后，章节内容会写入该文件夹中。
